# Windows 설치 파일 빌드 가이드

## 사전 요구사항

1. Node.js 설치 (v16 이상)
2. Windows 환경 (또는 macOS/Linux에서 크로스 컴파일)

## 빌드 방법

### 1. 의존성 설치

```bash
npm install
```

### 2. Windows 설치 파일 빌드

```bash
npm run build:win
```

이 명령어는:
1. React 앱을 빌드합니다 (`npm run build`)
2. Windows용 설치 파일(.exe)을 생성합니다

### 3. 빌드 결과물

빌드가 완료되면 `dist-electron` 폴더에 다음 파일들이 생성됩니다:
- `재정 입력 & 관리 프로그램 Setup x.x.x.exe` - 설치 파일

## 설치 파일 특징

- **NSIS 설치 프로그램**: 사용자가 설치 경로를 선택할 수 있습니다
- **바로가기 생성**: 데스크톱과 시작 메뉴에 바로가기가 자동 생성됩니다
- **자동 업데이트**: 추후 자동 업데이트 기능 추가 가능

## 배포

생성된 `.exe` 파일을 Windows 사용자에게 배포하면 됩니다.
설치 시 SQLite 데이터베이스는 사용자 데이터 디렉토리에 자동으로 생성됩니다.

## 참고

- 빌드 시간은 시스템 성능에 따라 다를 수 있습니다 (보통 2-5분)
- 첫 빌드 시 electron-builder가 필요한 바이너리를 다운로드하므로 시간이 더 걸릴 수 있습니다
